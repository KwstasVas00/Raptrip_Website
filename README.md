# Raptrip_Website

📌 Project Overview
This project is a website developed for a Greek business, in collaboration with [@dimk472](https://github.com/dimk472).
The website provides structured information about the company, its services, and contact details, with a focus on clean design and user-friendly navigation.

🌐 Live Demo
https://raptrip.gr

⚙️ Technologies Used
HTML
CSS
JavaScript

📱 Features
Responsive design (optimized for desktop, laptop, tablet, and mobile devices)
User-friendly interface (UI)
Navigation with internal hyperlinks
Contact form for user messages
Mobile optimization scripts for better performance and usability
