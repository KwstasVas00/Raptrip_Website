# Raptrip-Website

📌 Project Overview<br>
This project is a website developed for a Greek business, in collaboration with [@dimk472](https://github.com/dimk472).<br>
The website provides structured information about the company, its services, and contact details, with a focus on clean design and user-friendly navigation.<br>

🌐 Live Demo<br>
https://raptrip.gr<br>

⚙️ Technologies Used<br>
HTML<br>
CSS<br>
JavaScript<br>

📱 Features<br>
Responsive design (optimized for desktop, laptop, tablet, and mobile devices)<br>
User-friendly interface (UI)<br>
Navigation with internal hyperlinks<br>
Contact form for user messages<br>
Mobile optimization scripts for better performance and usability<br>
